🚀 通过GitHub部署到Vercel - 最可靠的方法

由于直接拖拽可能不太明显，我们用GitHub方式部署（更稳定）

═══════════════════════════════════════════════════

第1步：创建GitHub仓库
═══════════════════════════════════════════════════

1. 访问：https://github.com/new

2. 填写信息：
   Repository name: cyber-divination
   Description: 赛博神谕占卜系统
   选择：Public（公开）
   不要勾选任何初始化选项

3. 点击 "Create repository"

═══════════════════════════════════════════════════

第2步：上传文件到GitHub
═══════════════════════════════════════════════════

创建仓库后，GitHub会显示一个页面，找到：

"uploading an existing file" 这个链接，点击它

或者直接点击页面上的 "uploading an existing file"

然后：
1. 将 cyber-divination-deploy 文件夹中的所有文件
2. 拖到GitHub的上传区域
3. 点击 "Commit changes"

═══════════════════════════════════════════════════

第3步：在Vercel导入GitHub仓库
═══════════════════════════════════════════════════

1. 回到Vercel页面（你现在看到的页面）

2. 在 "Import Git Repository" 区域

3. 点击右上角的刷新按钮（如果看不到你的仓库）

4. 找到 "cyber-divination" 仓库

5. 点击 "Import"

6. 配置：
   - Project Name: cyber-divination
   - Framework: Other
   - 其他保持默认

7. 点击 "Deploy"

8. 等待1-2分钟

9. 完成！获得链接

═══════════════════════════════════════════════════

🔗 访问链接

https://你的项目.vercel.app/zb
