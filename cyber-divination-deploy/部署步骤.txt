🚀 Vercel 拖拽部署 - 3分钟搞定

═══════════════════════════════════════════════════

第1步：打开两个窗口
═══════════════════════════════════════════════════

窗口1 - 浏览器：
→ 访问 https://vercel.com/new
→ 用GitHub账号登录（点击 Continue with GitHub）

窗口2 - Finder：
→ 已经打开了（就是现在这个文件夹）
→ 看到这些文件：
  ✓ index.html
  ✓ vercel.json
  ✓ 占卜页面-付费完整版.html
  ✓ 占卜页面-免费体验版.html
  ✓ 占卜终极版.html
  ✓ 兑换码文件夹

═══════════════════════════════════════════════════

第2步：拖拽上传（最关键！）
═══════════════════════════════════════════════════

在浏览器中，你会看到一个虚线框，上面写着：
"Drop your project folder here"
或
"拖放项目文件夹到这里"

现在做这个动作：

1. 鼠标点击 Finder 中的 cyber-divination-deploy 文件夹
   （就是包含这个文件的文件夹）

2. 按住鼠标不放

3. 拖到浏览器的虚线框里

4. 看到虚线框变成蓝色或绿色

5. 松开鼠标

6. 等待上传（10-30秒）

═══════════════════════════════════════════════════

第3步：配置项目
═══════════════════════════════════════════════════

上传完成后，会看到配置页面：

Project Name（项目名称）：
→ 改成 cyber-divination
→ 或者保持默认也行

Framework Preset（框架）：
→ 选择 Other
→ 通常已经是默认的

其他设置：
→ 全部保持默认
→ 不用改

═══════════════════════════════════════════════════

第4步：点击 Deploy 按钮
═══════════════════════════════════════════════════

在页面底部：
→ 找到蓝色的 Deploy 按钮
→ 点击它
→ 等待 1-2 分钟

你会看到：
Building... → Deploying... → Ready ✓

═══════════════════════════════════════════════════

第5步：获取链接
═══════════════════════════════════════════════════

部署成功后，会显示：
🎉 Congratulations!

你的链接：
https://cyber-divination-xxxxx.vercel.app

→ 复制这个链接
→ 保存到备忘录

═══════════════════════════════════════════════════

🔗 访问你的网站
═══════════════════════════════════════════════════

首页：
https://你的项目.vercel.app

付费版（推荐用这个推广）：
https://你的项目.vercel.app/zb

免费体验：
https://你的项目.vercel.app/free

═══════════════════════════════════════════════════

🧪 测试
═══════════════════════════════════════════════════

1. 访问 /zb 链接
2. 输入梦境（随便写10个字以上）
3. 点继续
4. 输入测试码：TEST2024
5. 看到完整结果 = 成功！

═══════════════════════════════════════════════════

❓ 如果拖拽不行怎么办？
═══════════════════════════════════════════════════

方法1：点击 Browse 按钮
→ 在 Vercel 页面找到 Browse 按钮
→ 点击它
→ 选择 cyber-divination-deploy 文件夹
→ 点击打开

方法2：刷新重试
→ 刷新 Vercel 页面
→ 重新拖拽

═══════════════════════════════════════════════════

💡 关键点
═══════════════════════════════════════════════════

✓ 拖的是整个文件夹，不是里面的单个文件
✓ 要拖到虚线框里，不是页面其他地方
✓ 看到虚线框变色才松开鼠标
✓ 整个过程 3-5 分钟
