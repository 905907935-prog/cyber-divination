## 🎯 使用流程

### 发货流程

1. **收到订单**
   - 小红书店铺收到订单通知
   - 记录订单号和买家信息

2. **选择兑换码**
   - 打开CSV文件
   - 找到第一个"未使用"的兑换码
   - 复制兑换码

3. **发送给买家**
   ```
   您好!感谢购买~
   您的兑换码是: XHS2-CVW3
   
   使用方法:
   1. 访问占卜页面(见商品详情)
   2. 完成占卜仪式(选择塔罗牌)
   3. 输入兑换码查看完整结果
   
   有问题随时联系我哦~
   ```

4. **标记已使用**
   - 在CSV文件中将"使用状态"改为"已使用"
   - 记录订单号和使用时间
   - 保存文件
